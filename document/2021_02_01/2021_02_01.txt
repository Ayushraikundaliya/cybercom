#XML

-> Extensible Markup Language
-> To load xml file : $xml = simplexml_load_file('filename');

Xml file :

<producers>

	<producer>

		<name>Ayush</name>
		<age>22</age>

	</producer>

	<producer>

		<name>Nidhi</name>
		<age>20</age>

	</producer>

	<producer>

		<name>Mahi</name>
		<age>12</age>

	</producer>


</producers>


To access it thorugh php :

<?php

$xml = simplexml_load_file('example.xml');

foreach ($xml->producer as $producer) {
	echo $producer->name.' '.$producer->age.'<br>';
}

?>
